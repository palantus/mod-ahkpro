Shared lists is a form of todo lists for which you do not need a user.

The product is divived up into two concepts - buckets and lists. Buckets contains lists and lists contains the actual todos.

Every bucket and list is identified by an ID which can be assigned manually or chosen randomly

If you have given for instance a list the ID "newlist", then it can be accessed by anyone at ahkpro.dk/sharedlists?l=newlist.

If it is a bucket, then it can be accessed by swapping the l= with b=.

If you do not choose an ID, then a random one is selected for you (like ba8fe7a5-2667-f38e-213b-3c98e3f23ed6).

This means that you can share lists and buckets with anyone and only those who knows the ID can view and modify them!

This can be accessed on a phone as well and even be used offline (reading only).
